# Netty-Chat

------

**一个基于Netty开发的聊天系统，前端使用Jquery、Bootstrap搭建。此项目最适合Netty初级开发人员入手，加强对Netty的进一步掌握程度。该项目还可扩展并完善剩下的功能！！！**



### 功能实现

| 功能         |   是否已实现   |
| :----------- | :------------: |
| 公共信息发送 |      实现      |
| 公共信息接收 |      实现      |
| 用户名去重   |      实现      |
| 单对单信息   | 基础代码已编写 |
| 群发信息     | 基础代码已编写 |



### 使用方式

```tex
1、克隆此项目到本地
2、启动服务器(NettyServer.java)
3、打开html即可
```
